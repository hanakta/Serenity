React 18 + TypeScript

Tailwind CSS + Framer Motion

Lucide React иконки

Backend

PHP 8.1+

Slim Framework 4

JWT Authentication

RESTful API

База данных

SQLite

Встроенная БД

Таблицы: users, tasks, projects

Оптимизированные запросы

API Endpoints

Authentication: - POST /api/auth/register - регистрация - POST /api/auth/login - вход - GET /api/auth/me - профиль пользователя Tasks: - GET /api/tasks - список задач - POST /api/tasks - создание задачи - PUT /api/tasks/{id} - обновление задачи - DELETE /api/tasks/{id} - удаление задачи
